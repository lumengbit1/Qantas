# Qantas Program

## Getting started

#### Requirement

This app uses node v16.15.0, or pnpm v7.32.5

#### Project setup

Install project dependencies you can use npm:

```
pnpm install
```

#### Run development server

```
pnpm dev
```

#### Run linters

```
pnpm lint
```

#### Run tests

```
npm run test
```

#### Create production build

```
npm run build
```
